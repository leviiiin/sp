Sticky Password をより良い製品にするために、私たちは常に努力を続けています。そして今日、新たに追加された非接触接続機能をご案内できることを嬉しく思います。非接触接続は、ログインをこれまで以上に簡単かつ安全に行える、パワフルな新機能です。

 
 

パスワードの必要性は今後も存在し続けますが、その使い方は進化しています。

パスキーは ｢パスワード不要｣ のログイン方法として人気を集めていますが、利用できる機会はまだ限られており、パスワードは依然としてオンラインセキュリティにおいて重要な役割を果たしています。パスワードを安全かつ効率的に管理することは、これまで以上に重要になっています。

非接触接続機能を使えば、これまで以上にフレキシブルなパスワードへのアクセスが可能になります。非接触接続は Windows と macOS に加え、Linux、ChromeOS、仮想環境、リモートデスクトップ、リモート・ブラウザ・アイソレーションや、Sticky Password がインストールされていないデバイス上でも、多機能で安全なアクセスを可能にします。

インストールや煩雑なパスワードコピーも不要で、ログインはこれまでと同様に簡単かつ安全に行えます。

スマートフォンで QR コードをスキャンして認証するだけで、パスワードは必要な時、必要な場所だけに安全に入力され、常にエンドツーエンド暗号化が有効な状態に保たれます。

 
 

この機能は Sticky Password ユーザーに次のメリットをもたらします:

よりフレキシブルに—Sticky Password がインストールされていないコンピュータ上でも、どこでもログインできます。
安全性の高いブラウザなど多数のプラットフォーム上で機能—Windows、macOS、Linux、ChromeOSや、シークレットモード・プライベートモードが有効なブラウザからもシームレスにログインできます。
より速く、無駄のないログイン体験—タイピングや面倒な操作は必要ありません。
より強固なセキュリティ—ログイン情報の盗難やマルウェアからパスワードを保護します。
 
 

これは、Sticky Password が継続的に改善している方法の 1 つに過ぎず、オンラインでの安全を確保するために強力で使いやすい機能を提供します。
